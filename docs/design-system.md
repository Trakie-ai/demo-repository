# Trakie Design System

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| Primary Green | `#2D6A4F` | Buttons, active states, status indicators |
| Light Green | `#40916C` | Headings, hover states, accents |
| Background Primary | `#0A0A0A` | Page background |
| Background Card | `#1A1A1A` | Card surfaces, modals |
| Border | `#2A2A2A` | Dividers, card borders |
| Text Primary | `#F5F5F5` | Body text, headings |
| Text Secondary | `#A0A0A0` | Labels, captions, placeholders |

## Typography

| Role | Font | Weight | Size |
|------|------|--------|------|
| Sans (UI) | Outfit | 300-700 | — |

## Dark Theme

Trakie uses a dark-first design. All surfaces default to dark backgrounds. There is no light mode.
